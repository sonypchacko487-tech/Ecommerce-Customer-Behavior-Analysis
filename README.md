# Ecommerce-Customer-Behavior-Analysis
Data Analysis Project using Python

## 📊 Project Overview
This project analyzes customer behavior in an e-commerce platform using Python.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📌 Key Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Customer Behavior Insights
- Business Recommendations

## 📈 Insights
- Customer engagement impacts purchases
- High cart abandonment reduces revenue
- Churn is linked to low activity

## 💡 Conclusion
Improving engagement and retention strategies can significantly increase revenue.
